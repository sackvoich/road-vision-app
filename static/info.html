Как сделаны страницы

Фронтенд сверстан без фреймворков (чистые HTML/CSS/JS). Структура:

/static
  ├─ index.html
  ├─ info.html
  ├─ people.html
  ├─ css/styles.css
  └─ js/
     ├─ app.js
     └─ info.js
  • index.html: камера через getUserMedia, холст canvas, кнопки снимка и загрузки, отправка кадра по существующему WebSocket /ws/video.
  • info.html и people.html: статические страницы, общий стиль, лёгкие улучшения UX через info.js.
  • styles.css: адаптивная сетка, карточки, единый стиль элементов управления.

Технологии и модели

  • Фронтенд: HTML, CSS, JavaScript, getUserMedia, <canvas>, WebSocket.
  • Взаимодействие: WebSocket с сервером по адресу /ws/video, отправка кадров в формате Base64 JPEG.
  • Бэкенд: FastAPI, Uvicorn, обработка двух моделей в параллельных потоках.
  • Модели: Ultralytics YOLO (детекция знаков и сегментация «зебры»), техники: детекция, сегментация, постпроцессинг боксов/масок.
  • Версии из проекта: fastapi, uvicorn, ultralytics, torch, torchvision, torchaudio, coremltools, opencv-python-headless.
